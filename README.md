# Stroke-Prediction - 腦中風預測

## 動機
在台灣，腦中風一直是位居國人十大死因中的二到四名，每年發生的中風人數約3萬人。即使治癒了，將近半數以上的中風病人會殘留輕重不一的失能症狀。因此，如果能提早進行中風預防篩檢，找出中風高風險除群加以健康管理及治療，可以減少中風的發生率及致殘率。這樣不但減少個人的病痛，更可減輕家庭的重大負擔，同時也可節省國家長照醫療資源。

## 研究方法簡介
1. 資料預處理
    - 檢查並刪除缺失值及不必要的資料
    - 資料型別轉換
    - 特徵選取
    - 將資料進行分析統整，觀察數據是否有偏差
2. 使用四種 AI model 進行預測分析
    - Random Forest
    - Gaussain Naïve Bayes
    - DecisionTree
    - ANN
